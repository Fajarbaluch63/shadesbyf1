# Thread & Ledger — Abaya Business Tracker

Full-stack app: Node.js + MongoDB Atlas + Express

## Files
- server.js       — Express API server
- public/index.html — Full frontend (all tabs)
- package.json    — Dependencies
- .env            — Local config (do NOT commit to GitHub)

## Local Development
```
npm install
node server.js
# Open http://localhost:3000
```
